After exiting the configuration menu launch dump tool and dump your nand to your sd card

then reboot and redo the exploit to relaunch tm++

Run the unlaunch installer and select install unlaunch

After that press and hold the A and B buttons and press the power button once

After it boots into unlaunch select option, and change the boot preferences. I have no button set to tm++ and a set to launcher
